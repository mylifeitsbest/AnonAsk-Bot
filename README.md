# Telegram Bot for Anonymous Communication (Deep-linking API)

Асинхронно-потоковый Telegram-бот для организации защищенных анонимных диалогов между пользователями с использованием механизмов Deep-linking. Проект спроектирован с упором на минимизацию накладных расходов памяти и автоматическое управление жизненным циклом сессий.

## 🚀 Ключевой функционал и архитектурные решения

* **Механизм Deep-linking**: Генерация уникальных реферальных ссылок на базе Telegram User ID для инициации диалога без раскрытия персональных данных.
* **Многопоточное управление сессиями (TTL)**: Использование модуля `threading` для выполнения фонового демона (`garbage collector`), который ежечасно очищает оперативную память от истекших сессий (таймаут — 24 часа). Это предотвращает утечки памяти (`memory leaks`).
* **Двунаправленная маршрутизация сообщений**: Динамическая привязка `message_id` и `CallbackQuery` для обеспечения бесшовного ответа анонимному собеседнику в один клик через инлайн-клавиатуру.
* **Ролевая модель (RBAC)**: Реализован базовый функционал контроля доступа для администраторов (проверка по списку `ADMIN_IDS`), позволяющий проводить сквозной мониторинг входящего трафика в целях модерации.
* **Унифицированное логирование**: Собственная легковесная система логирования событий в stdout с фиксацией таймстампов, типов событий и идентификаторов сессий для упрощения дебага.

## 🛠 Стек технологий

* **Runtime**: Python 3.10+
* **Framework**: `pyTelegramBotAPI` (`telebot`) — синхронно-блокирующая реализация Long Polling.
* **Core Modules**: `threading` (параллельные фоновые процессы), `datetime` (валидация TTL).

## 📊 Схема работы системы

1. **Инициализация**: При вызове команды `/start` без аргументов система генерирует URL вида `t.me/username_bot?start=USER_ID`.
2. **Установка соединения**: Переход по ссылке с аргументом парсит ID целевого пользователя и создает в памяти (структура данных `dict`) две зеркальные записи с временной меткой `expires`.
3. **Проксирование**: Все последующие текстовые сообщения инкапсулируются, снабжаются контекстной инлайн-кнопкой для ответа и пересылаются адресату.

## ⚙️ Инструкция по развертыванию

### 1. Окружение
Для работы скрипта необходим интерпретатор Python версии 3.10 или выше и установленный пакет `pyTelegramBotAPI`:

```bash
pip install pytelegrambotapi
```

### 2. Конфигурация
Перед запуском необходимо прописать системные переменные внутри исполняемого файла:
* `API_TOKEN`: Токен бота, полученный от BotFather.
* `ADMIN_IDS`: Список целочисленных Telegram ID учетных записей, обладающих правами суперпользователя.

### 3. Запуск сервиса
```bash
python main.py
```
