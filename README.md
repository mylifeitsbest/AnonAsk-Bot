🕵️‍♂️ AnonAsk Bot (aka "Узнай всю правду о себе")Очередной (но самый четкий) Telegram-бот для анонимных вопросов и ответов, написанный на pyTelegramBotAPI (telebot). Отлично подходит для того, чтобы закинуть ссылку в шапку профиля и узнать, что о тебе думают бывшие, друзья или коллеги.🔥 Киллер-фичи (Почему этот бот — топ)Режим "Админ видит всё": Если твой ID в списке админов, анонимность ломается. Ты будешь видеть юзернеймы тех, кто тебе пишет. Полный тоталитаризм.Автоматический клининг: Диалоги живут ровно 24 часа. Бот запускает отдельный поток-киллер (threading), который раз в час выметает просроченные переписки.Удобные инлайн-ответы: Получателю не нужно гадать, куда писать ответ. Под сообщением появляется кнопка ✉️ Ответить, которая автоматически связывает диалог обратно.🛠 Стек технологийЯзык: Python 3.xФреймворк: pyTelegramBotAPI (telebot)Архитектура: Threading (для фоновой очистки памяти), Логирование в реальном времени.🚀 Быстрый старт1. Используйте код с осторожностью.📋 Как это работает (Под капотом)[Пользователь А] ---> (Генерирует ссылку /start=ID) ---> [Делится в соцсетях]
[Пользователь Б] ---> (Переходит по ссылке) ---> [Пишет боту] 
Бот ---> [Пересылает сообщение Пользователю А анонимно] 
/start — без параметров выдает твою личную реферальную ссылку. С параметром (ID другого юзера) — открывает портал анонимного общения на 24 часа.Текстовые сообщения — бот проверяет, находишься ли ты в активном диалоге, и аккуратно перекидывает месседж твоему собеседнику, вешая инлайн-кнопку.⚠️ ДисклеймерАвтор не несет ответственности за душевные травмы, полученные в ходе чтения анонимных сообщений.
