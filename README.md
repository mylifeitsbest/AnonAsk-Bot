# AnonAsk-Bot

Простой Telegram-бот на библиотеке `pyTelegramBotAPI` (`telebot`), который позволяет пользователям обмениваться анонимными сообщениями с помощью специальных ссылок.

## 🚀 Основной функционал

* **Генерация ссылок**: При первом запуске бот выдает пользователю уникальную ссылку. Тот, кто перейдет по ней, сможет написать анонимное сообщение.
* **Система ответов в один клик**: Под каждым анонимным сообщением появляется инлайн-кнопка «Ответить». Получатель может сразу написать ответ, не раскрывая личность отправителя.
* **Ограничение времени диалога (TTL)**: Созданный диалог активен ровно 24 часа. Бот автоматически удаляет устаревшие сессии из памяти каждый час, чтобы не перегружать сервер.
* **Функционал администратора**: В коде можно указать ID администраторов. Они будут видеть юзернеймы пользователей, которые отправляют сообщения через бота.
* **Логирование**: Бот выводит в консоль историю всех действий (генерация ссылок, отправка сообщений и очистка памяти) с указанием точного времени.

## 🛠 Стек технологий

* **Язык**: Python 3.x
* **Библиотека**: `pyTelegramBotAPI` (`telebot`)
* **Дополнительно**: `threading` (для фоновой очистки памяти), `datetime` (для контроля времени жизни диалогов).

## ⚙️ Установка и запуск

### 1. Установка библиотеки
```bash
pip install pytelegrambotapi
```

### 2. Настройка
Вставьте ваш токен и ID администраторов в переменные в начале файла:
```python
API_TOKEN = 'ВАШ_ТОКЕН'
ADMIN_IDS = # ID администраторов
```

### 3. Запуск
```bash
python main.py
```
